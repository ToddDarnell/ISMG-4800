Dynamic Styles

  • Dynamic styles allow you to dynamically set different style attributes for the document.
  • For example: the function changeBackground prompts the user to enter an image name and then sets the background to that image.  (Note: the first function does not work if the original background image was set using CSS).

        

<style>
  .body {background-image: url("../../images/background/hearts.gif");}
</style>
