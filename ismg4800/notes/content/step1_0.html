• In order to make an HTTP request to the server using JavaScript, you need an instance of a class that provides you this functionality.
    •  XMLHTTP was originally introduced for Internet Explorer
    • Then Mozilla, Safari etc. then implemented the XMLHttpRequest class.
  • To create a cross-browser instance (object) of the required class, you: 
 if (window.XMLHttpRequest) { // Mozilla, Safari,...
     http_request = new XMLHttpRequest();
     if (http_request.overrideMimeType) {
         http_request.overrideMimeType('text/xml'); // To fix error 
     }                                              // in some browsers
 } else if (window.ActiveXObject) { // IE
     try {  http_request = new ActiveXObject("Msxml2.XMLHTTP");
     } catch (e) {
         try { http_request = new ActiveXObject("Microsoft.XMLHTTP");
         } catch (ex) {}
     } }
