• AJAX (Asynchronous JavaScript and XML) is a new term for two powerful browser features that have been around for years, but were overlooked by many web developers until recently when applications such as Gmail, Google Suggest, and Google Maps hit the streets.
  • The two features that make AJAX a new way to look at Web design are that you can:
    • Make requests to the server without reloading the page
    • Parse and work with XML documents
