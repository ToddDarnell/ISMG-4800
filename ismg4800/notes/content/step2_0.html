• When you send the AJAX request, you provide the name of a JavaScript function that handles the response.
       http_request.onreadystatechange = nameOfTheFunction;
    This function needs to check for the state of the request.
  • The full list of the readyState values is as follows:
    0 (uninitialized)        
    1 (loading)
    2 (loaded)     		3 (interactive)
    4 (complete)
    If the state has the value of 4, that means that the full server response is received and it's OK for you to continue processing it. 
if (http_request.readyState == 4) {
    // everything is good, the response is received
    // so do something
}
