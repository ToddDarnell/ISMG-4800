Dynamic HTML: Filters and Transitions (IE Only)

  • Applying filters to text and images causes changes that are persistent. You can convert colored images to gray in response to user actions. You can make letters glow for emphasis. You can create drop shadows to give text a three-dimensional appearance.
  • Transitions are temporary phenomena: applying a transition allows you to transfer from one page to another with a pleasant visual effect such as a random dissolve. 
  • Filters and transitions do not add content to your pages-rather, they present existing content in an engaging manner to help hold the user's attention. 
  • For Example text can be flipped:

Drawing

  • You can create original artwork for your Flash animations using the Drawing and Painting tools.
  • These tools are very standard, with basic stapes, lines fills etc.
  • The Drawing lesson provided by Flash steps you through using all the different drawing tools.

Symbols

  • A symbol in Flash is a reusable image, animation, or button.
  • An instance is an occurrence of a symbol on the Stage (main work area).
  • Symbols help you keep Flash file sizes low because they reuse as much information as possible from a single Symbol and simplify the editing of Flash movies.
  • Symbols are also required to create sophisticated interactivity.
  • In the demonstration animation all the pieces of the animation were separate symbols (top of egg, rectangle, arrows...). 
  • Symbols are similar to using "Group" in Power Point.

Layers

  • You can think of layers as transparent sheets of acetate stacked on top of each other.
  • Where there is nothing on a layer, you can see through it to the layers below. Layers help you organize the artwork in a Flash movie.
  • A Flash object that is on a top layer will appear above a Flash object on a lower layer.
  • You can draw and edit artwork on one layer without affecting artwork on another layer.
  • The screenshot of the Flash development environment shows all the layers I had in my Flash demonstration.

Type

  • You can add type to your movies and set its size, typeface, style, spacing, color, and alignment.
  • You can also manipulate type like an object or as basic shapes to create sophisticated effects.
  • In the type lesson provided by Macromedia, you'll learn how to:
    • Add type to a movie, format it, and break it into shapes.
    • Create text fields you can use to add interactivity to a movie.

Buttons

  • Buttons let viewers interact with Flash movies. For example, the Next button in this movie changes the Stage by advancing the playhead to the next scene.
  • To create a button, you create images for the Up, Over, and Down states of the button. Generally, these images are slight variations of one image.
  • In addition, you also specify the area that triggers the button interaction, called the Hit state. When the pointer clicks or rolls over a button, Flash displays the appropriate image.

Sound

  • Sound can be an integral part of a Flash movie.
  • There are two types of sounds in Flash:
  • Streaming sounds:
    • Streaming sounds play as long as the layer containing the sound continues in the Timeline.
    • Streaming sounds are best for long background sounds.
    • When a movie is downloading from an Internet source, a streaming sound can begin playing as soon as a small portion of the sound has been downloaded.
  • Event Sounds:
    • Event sounds continue playing once it starts, no matter what happens in the movie.
    • An event sound is best for short sounds, because it must download completely and load into RAM before it begins playing.
    • Events sounds are most useful for buttons.
    • In many cases you want a button to move the movie to a new scene, but you don’t want the button sound to stop playing abruptly when the scene switches, as it would if you use a streaming sound.

Animation

  • Tweened animation is an essential technique for creating movement and change in a movie while minimizing file size.
  • In my example, objects moved across the "Stage", changed in size, and rotated.
  • You can also make objects change color and change shape.
  • Animation in flash uses a concept called keyframes.
    •  A keyframe is a frame where you specify changes in the animation.
    • In tweened animation, you create keyframes at important points in the Timeline and let Flash create the frames in between.
